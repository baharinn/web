# Galatasaray Maç Merkezi

Bu proje, Hacettepe Üniversitesi Eğitim Fakültesi  
Bilgisayar ve Öğretim Teknolojileri Eğitimi (BÖTE)  
Web Programcılığı dersi final ödevi kapsamında geliştirilmiştir.

## Kullanılan Teknolojiler
- React (Create React App)
- Fetch API
- API-FOOTBALL

## Kullanılan API
https://www.api-football.com/

## Proje Özellikleri
- Galatasaray’ın Süper Lig maç sonuçlarını listeler
- Kazanılan, kaybedilen ve berabere biten maçları filtreleme
- Sonraki maç bilgisini otomatik gösterme
- Oyuncu kartları ve detay popup ekranı
- Loading ve hata mesajları
